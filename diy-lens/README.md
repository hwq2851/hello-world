# 400mm DIY 镜头 - 3D 打印零件

松下 S9（L 卡口） × 80mm f/5 双合透镜 = 400mm 手动镜头

## 文件说明

| 文件 | 用途 |
|------|------|
| `part1_lens_ring.scad` | 物镜压环：固定 80mm 镜片到 PVC 管前端 |
| `part2_l_mount_adapter.scad` | L 卡口转接座：连接镜筒到相机 |
| `part3_focusing_tube.scad` | 调焦内管：前后滑动微调合焦 |
| `all_parts_preview.scad` | 预览全部零件（打开此文件可同时查看） |

## 使用方法

1. 下载安装 [OpenSCAD](https://openscad.org/)（免费开源）
2. 打开 `all_parts_preview.scad` 预览效果
3. 分别打开 `part1`、`part2`、`part3`，按 F6 渲染后导出 STL
4. 把 STL 文件发给淘宝 3D 打印店

## 打印参数建议

| 参数 | 推荐 |
|------|------|
| 零件 1（压环） | PLA 黑色，填充 40%，层高 0.2mm |
| 零件 2（转接座） | **ABS 黑色**（更耐久），填充 50%，层高 0.2mm |
| 零件 3（内管） | PLA 黑色，填充 30%，层高 0.2mm |

## 总装结构

```
[物镜 80mm] → [压环] → [PVC 管 Φ90mm] → [调焦内管] → [L卡口转接座] → [松下 S9]

总长 ≈ 400mm + 调焦余量
```

## 注意事项

- L 卡口法兰距 = 20mm，需精确保证
- 3D 打印尺寸可能有 ±0.2mm 误差，建议先打印测试件
- 组装前在内壁贴黑色植绒纸防炫光
